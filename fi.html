About

Are you a Problem Author?
The toughest part of creating a problem is creating tricky, correct and constrained Test Cases.
Well, Here is 'One Click Python Test Case Generator', for your respective logic(solution).

Test Case Generator is a tool which helps problems authors to generate test cases if given solution file and input format and constraints.
This all happen in tcgen.py file, the print section inside the file opening loop is what gets printed inside inputXX.txt file and then these files are used as input for logic file and then output generated in

Steps

How to Use?

  • Clone the repository $ git clone https://github.com/aashutoshrathi/HackerRank-Test-Case-Generator.git
  • Create a virtual environment $ virtualenv venv , click here to read about activating virtualenv
  • Activate virtualenv (Linux) $ source ./venv/bin/activate
  • Activate virtualenv (Windows) $ cd venv/Scripts/
    $ activate
    $ pip install -r requirements.txt
  • Go to sh tc_generator and run the project. $ cd tc_generator/
    $ python tc_gen.py
